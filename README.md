BはAより大きい  
AはCより小さい  
BはCより小さい  
最も小さいのは?  

といった感じのQuizをドンドン出題してくるアプリ。  
日本語,繁體中國語,韓國語,英語に對應。

### Test環境(Test Environment)

- Python 3.8.1 + Kivy 2.0.0
